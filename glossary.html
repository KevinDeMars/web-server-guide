Glossary

Apache
Name of the web server we use in this guide.
CSS
"Cascading style sheet". Determines how a page looks (colors, font sizes, borders, spacing, etc.) See Putting Content on the Server.
Document root
The top-level folder where you put files to serve. By default, this is "C:\Apache24\htdocs". See Configuration.
Domain name
Essentially the name of a website, such as "example.com".
Domain registrar
An organization that can assign people domain names (for a price). Also associates domain names with an IP address.
htdocs
The default folder name for the document root in Apache.
HTML
"Hypertext Markup Language". See Putting Content on the Server.
HTTP
"Hypertext Transfer Protocol". See Introduction.
IP address
A number that identifies a computer; is private or routable/public. See Introduction.
Network
A group of computers "next to" each other, all connected to a router. See Introduction.
Port
Kind of like a TV channel but for internet traffic. See Introduction.
Port forwarding
Tells your router to give your computer HTTP traffic instead of throwing it away. See Introduction and Configuration.
Private address
An IP address that is only valid within a single network. See Introduction.
Public address
Synonym of "routable address".
Routable address
An IP address that is globally unique and usable accross networks. See Introduction.
Router
Allows networks to talk to other networks. See Introduction.
Server root
Location where you extracted Apache's main folder. Defaults to "C:\Apache24". See Configuration.
URL
For our purposes, is the location of a specific file (e.g. "example.com/hello/file.txt")
