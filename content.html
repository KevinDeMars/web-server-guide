Putting Content on the Server

Static files

Although the main purpose of a web server is to host web pages, you can host any kind of file. Since the files sent by the server will be the same every time, this is called static file hosting.

To share a file, simply place it inside the "htdocs" folder (or a subfolder within it). You can share images, text, or any other arbitrary binary file.

Procedure

  1. Create a text file called "my-message.txt" with a message of your choice.
  2. Save it inside your "htdocs" folder.

HTML editing

To host a web page, the simplest way is to write a .html file and put it in the "htdocs" folder. Your web browser is smart enough to render the HTML as a fancy-looking web page rather than a bunch of text with tags.

What does a .html file look like? It is mostly plain text, but there are tags -- identified by angle brackets -- around the text that specify how to interpret the text (e.g. "this part is a header"). Tags can be nested inside each other. A piece of text is surrounded with an opening tag (<p>) and a matching closing tag (</p>)

A page has two parts, a header and a body. For the purpose of this guide, the header is just for specifying the page title or embedding scripts and stylesheets. The main content is in the body.

Procedure

Create a file in your "htdocs" folder called "my-page.html" with the following contents:

<!DOCTYPE html>
<html lang="en">
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello!</h1>
    <p>This is a page that demonstrates the basics of HTML editing:</p>

    <ul>
        <li>Page layout (head and body)</li>
        <li>Header tags (h1)</li>
        <li>Paragraph tags (p)</li>
        <li>Bulleted lists (ul for "unordered list", li for "list item")</li>
        <li>Hyperlinks (a for "anchor")</li>
        <li>Images</li>
    </ul>

    <p>Click <a href="my-message.txt">here</a> for a cool message.</p>

    <p>Here is a cool picture:</p>
    <img src="my-picture.png" />
</body>
</html>

This page includes an example of adding a large header, adding a bulleted list, adding a link to another page, and adding an image.

The header is specified as <h1> because there are multiple levels of headers and we want the largest one, a level-1 header.

To make the image work, include an image in the same folder that you saved the HTML file to called "my-picture.png". The <img /> tag is special because it is a self-closing tag. You don't need a </img> after it.

Now, you should be able to see the page by going to localhost/my-page.html. Go here to see what it should look like.

CSS editing

The page you created in the last step looks quite plain. Luckily, we can make it fancy by adding some CSS declarations. CSS stands for Cascading Stylesheet and is used to specify how the things on the page should look.

A .css file is a list of declarations that specify what things they apply to, and what styles should apply to those things. For example, the following says to make all level-1 headers bold and underlined: 

h1 {
    font-weight: bold;
    text-decoration: underline;
}

Instead of saying "h1 {", we could say to apply these styles to anything with the ID of "myId" by writing "#myId {", or anything with the class "myClass" with ".myClass {". We can also specify more complicated things like "a paragraph directly contained in something with the ID of myId" by writing "#myId p {".

Let's apply some styling to the above page by adding some colors and a rounded container around the content (to make the page look like this one).

Procedure

  1. Change "my-page.html" to the following: 
    <!DOCTYPE html>
<html lang="en">
<head>
    <title>My Page</title>
    <link rel="stylesheet" href="my-stylesheet.css" />
</head>
<body>
    <div id="container">
        <h1>Hello!</h1>
        <p>This is a page that demonstrates the basics of HTML editing:</p>
    
        <ul>
            <li>Page layout (head and body)</li>
            <li>Header tags (h1)</li>
            <li>Paragraph tags (p)</li>
            <li>Bulleted lists (ul for "unordered list", li for "list item")</li>
            <li>Hyperlinks (a for "anchor")</li>
            <li>Images</li>
        </ul>
    
        <p>Click <a href="my-message.txt">here</a> for a cool message.</p>
    
        <p>Here is a cool picture:</p>
        <img src="my-picture.png" />
    </div>
</body>
</html>
    This does the following:
    • Puts all of the content inside a "div" container, and gives it an ID called "container".
    • Tells the page to look for a stylesheet called "my-stylesheet.css".
  2. Create a new file in the same folder called "my-stylesheet.css" with the following content:
    3. body {
    background-color: #ddd;
    font-family: 'Trebuchet MS', 'Lucida Sans Unicode', 'Lucida Grande', 'Lucida Sans', Arial, sans-serif;
    font-weight: bold;
    text-decoration: underline;
}

#container {
    background-color: #fff;
    width: 50%;
    margin: auto;
    border-radius: 1em;
    padding: 1em;
}

h1 {
    text-align: center;
}

img {
    object-fit: contain;
    max-width: 75%;
}

li {
    margin: 0.5em 0 0 0.5em
}
